# IRC & Discord

Most communication is done on our IRC or through the CompSoc Discord. These two channels are bridged together, so you can use whichever you'd prefer.

# IRC

If you've never used IRC before, look here (opens new window) for a quick tutorial.

# Connecting

Server: irc.imaginarynet.uk

Port: 6697 (TLS - tick 'use TLS/SSL' in your client)

Channel: #tardis

# Clients

If you have a tardis account, both irssi and weechat are installed on fez and torchwood.

You can also use our web client hosted at https://shout.tardis.ed.ac.uk (opens new window)

# Other Clients

There are a range of IRC clients most of which take some getting used to

  • here are some recommendations:

# Cross Platform

  • Pidgin (opens new window) - Does just about everything chat-wise.

# Windows

  • mIRC (opens new window) - The most popular IRC client (free but you get messages on startup asking you to pay)

# Linux

  • XChat (opens new window) - Windows version isn't free but a nice GUI linux chat client.
  • irssi (opens new window) - Console based IRC client, clean and highly configurable.
  • weechat

# Web-based

  • KiwiIRC (opens new window) - Pleasant browser-based chat client; however most native clients are more reliable and should be used when possible.
  • mibbit (opens new window) - Another browser-based chat client.

We use the CompSoc Discord server, which can be found here (opens new window)
